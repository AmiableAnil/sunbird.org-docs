The topic selector ensures that the content is linked with the topics and extracts correct metadata. The list of the topics appears according to the board, grade, subject, and medium that you have selected.

Step Screen
1 Click Topic. The Topics page is displayed
2. Search for a topic using the Search bar
3. The Topics tab displays a complete list of available topics
4. Browse and select the topic(s) from the topic tree
5. View the selected concepts in Selected Topics tab
6. Click Done to proceed
7. Click Cancel to exit
