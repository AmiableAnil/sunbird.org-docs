Overview

A Library typically refers to a space, housing a large collection of books, resources or information. Extending the literary meaning, Library, a key module of Sunbird, displays all content created or uploaded on the platform. The library is accessible to all registered users.

This page elaborates the various user interface elements of the Library

Prerequisites

Step Screen
1. You are logged in
2. You want to view the library, you have clicked on the Library tile

Searching for Content

Step Screen
Search for content using either of the following:
 a) Search bar
 b) Filter options, or
 c) Sort options
1. Select any of the following from the drop-down menu:
 a) All: To search for all content available in the portal
 b) Courses: To search for all the courses available in the portal
 c) Library: To search for content available in the library
 d) Organizations: To search for organizations uploaded in the portal
 e) Users: To search for users uploaded in the portal
2. Alternatively, you can type keywords in the search bar

Applying Filters

Step Screen
1.Click Filters
2. Select the following from their respective drop-down menus:
 a) Board
 b) Grade
 c) Subject
 d) Medium
 e) Content Types
 f) Concepts
2. Click Apply. The search results are displayed
3. Click Reset to reset the filters

Sorting Content

Step Screen
1. Click the Sort drop-down icon
2.Sort content on the:
 b) modified date, or
 b) created date

Viewing Popular Stories

Sunbird houses a wide variety of content. Users with registered credentials can view and download content from the Library.

Step Screen
1. Click on any story
2. Scroll right or left using the Arrows to scroll through listed content

Consuming a Story

Step Screen
1. Click the story you wish to consume. The story is displayed within the inbuilt content player. Description and content information of the story are provided below the player

Sharing Stories

Step Screen
1.Click the Share icon to share the story
2. A screen with the link to the story is displayed
3. Click COPY LINK to copy the link
4. Click Close icon to go back to the content player

Flagging Stories

Step Screen
1.Click Flag icon to flag the displayed content
2.Select a reason to describe why the content should be reviewed again
3. Add additional comments, if any
4. Click Submit to submit the reasons for flagging the content
5. Click Close icon to exit the page

Copying Content

Step Screen
1. Select any content from the Library to view the content
Note: Users with creation rights can copy any content available in the Library
1. Click Make Copy icon to copy content
The content is copied into the Draft folder.
Note: The following automatically happens to all content that is copied:  
  • All copied content resides in the Draft folder
  • The copied content is tagged with the prefix Copy of before the original content name
  • The user, organisation details, copy type, and source information are captured
  • The links to the assets and objects remain intact

    Taking Notes

    Step Screen
    1. Click TAKE NOTE icon or link to take notes while viewing the story
    2. Enter Title
    3. Enter Description
    4. You can format the description entered with the various formatting options available
    5. Click Clear to clear the note
    6. Click Save to save the note
    7. Click Close icon to exit the page
